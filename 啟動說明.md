# 如何啟動網站

## 問題說明
直接打開 `public/index.html` 會是空白的，因為這是 React 應用，需要通過 Webpack 編譯才能運行。

## 解決步驟

### 1. 安裝依賴
```bash
npm install
```

### 2. 啟動開發伺服器
```bash
npm run dev
```
或
```bash
npm start
```

### 3. 打開瀏覽器
開發伺服器會自動在 `http://localhost:3000` 啟動，如果沒有自動打開，請手動在瀏覽器中訪問該網址。

## 注意事項
- 不要直接打開 HTML 文件，必須通過開發伺服器運行
- 開發伺服器會自動監聽文件變化並重新編譯
- 按 `Ctrl+C` 可以停止開發伺服器

## 生產環境構建
如果需要構建生產版本：
```bash
npm run build
```
構建完成後，文件會在 `dist/` 目錄中。
